# Sales Analytics Dashboard using Power BI  

## Overview  
This project presents an interactive **Sales Analytics Dashboard** built using Power BI to analyze the performance of a single organization. The dashboard provides insights into **sales trends, profit, units sold, product performance, and regional contributions**, enabling stakeholders to make data-driven business decisions.

The report includes high-level KPIs, time-based trend analysis, and detailed product-level breakdowns with interactive navigation.

## Tools & Technologies  

| Tool | Purpose |
|-----|--------|
| **Power BI Desktop** | Dashboard creation & data visualization |
| **Power Query** | Data cleaning and transformation |
| **DAX** | KPI calculations and time intelligence measures |
| **GitHub** | Project documentation and version control |

## Dashboard Contents  
- Sales Overview page with KPIs and trends  
- Country-wise and segment-wise performance analysis  
- Product-level analysis by **Net Sales** and **Orders**  
- Interactive slicers and navigation buttons  

## Key Visualizations  
- **KPI Cards**: Total Sales, Net Profit, Units Sold, LYTD Sales, MTD Sales  
- **Net Sales Trend** (Year & Month)  
- **Top-10 Countries by Net Sales**  
- **Net Sales by Segment**  
- **Top-10 Products by Orders**  
- **Top-10 Products by Net Sales**

## Sample Insights  
- Sales exhibit **clear monthly trends and seasonality**  
- Government and Small Business segments generate the highest net sales  
- USA, Canada, France, and Germany are the top revenue-contributing countries  
- A small number of products contribute a majority of total sales and orders  
- Discounts significantly impact profit margins across regions  

## How to View the Dashboard  
1. Download the `.pbix` file from this repository  
2. Open it using **Power BI Desktop**  
3. Use slicers to filter by country and segment  
4. Navigate between pages using interactive buttons
